Focus Mode Chrome Extension 🚀

A simple and clean Chrome Extension that helps users stay productive by blocking distracting websites and providing a focus timer.

✨ Features
✅ Block distracting websites
✅ Simple Pomodoro focus timer
✅ Clean dark mode UI
✅ Lightweight and fast
✅ Beginner-friendly project

📌 How It Works
Website Blocking

The extension detects distracting websites and redirects or hides content to help users stay focused.

Focus Timer

Users can start a simple Pomodoro timer directly from the extension popup.

Dark UI

Minimal dark-themed interface for a modern productivity experience.
